\questao Considere os seguintes an{\'e}is: $(\real, +, \cdot)$ e $(\real, \oplus, \otimes)$, sendo $a \oplus b = a + b + 1$ e $a \otimes b = a + b + ab$. Mostre que $f : \real \to \real$ dado por $f(x) = x + 1$, para todo $x \in \real$, {\'e} um homomorfimo de $(\real, \oplus, \otimes)$ em $(\real, +, \cdot)$.

\noindent\textbf{Solu\c{c}\~ao:}
Precisamos mostrar que
\begin{enumerate}
	\item $f(x \oplus y) = f(x) + f(y)$
	\item $f(x \otimes y) = f(x)\cdot f(y)$
\end{enumerate}
para todos $x$, $y \in \real$.

De fato, para $x$, $y \in \real$ temos
\begin{align*}
	f(x \oplus y) &= f(x + y + 1) = (x + y + 1) + 1 = (x + 1) + (y + 1) = f(x) + f(y)\\
	f(x \otimes y) &=  f(x + y + xy) = (x + y + xy) + 1 = x(1 + y) + (1 + y) \\ &= (x + 1)(y + 1) = f(x)\cdot f(y)
\end{align*}
Logo $f$ é um homomorfismo de $(\real, \oplus, \otimes)$ em $(\real, +, \cdot)$.

\questao Considere o anel $\z$. O conjunto
\[
	B = \left\{ \dfrac{a}{2^n} \mid a \in \z,\ n \in \z\right\}
\]
é um subanel de $\z$?

\noindent\textbf{Solu\c{c}\~ao:} Primeio fazendo $a = 0 \in \z$ vemos que $0 \in B$. Logo $B \ne \emptyset$. Dados $x$ e $y \in B$ existem $a$, $b$, $m$ e $n \in \z$ tais que
\[
	x = \dfrac{a}{2^m} \quad \mbox{e} \quad y = \dfrac{b}{2^n}.
\]
Assim
\begin{align*}
	x - y &= \dfrac{a}{2^m} - \dfrac{b}{2^n} = \dfrac{a2^n - b2^m}{2^m2^n}\\
	xy &= \dfrac{a}{2^m}\dfrac{b}{2^n} = \dfrac{ab}{2^{m + n}}
\end{align*}
como $a2^n - b2^m \in \z$ e $ab \in \z$ segue que $x - y \in B$ e $xy \in B$ para todos $x$, $y \in B$. Portanto $B$ é um subanel de $\z$.
